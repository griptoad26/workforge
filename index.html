Explain the difference between useMemo and useCallback with a real example.
Grok · 2s ago
Both memoize, but for different things. useMemo caches a computed value:

const sorted = useMemo(() => items.sort(), [items]);

useCallback caches a function reference:

const handleClick = useCallback(() => doThing(id), [id]);

Use useMemo when a calculation is expensive. Use useCallback when passing callbacks to memoized children to prevent re-renders.

Frequently asked

Is my data really private?
Yes. XKG stores everything in a SQLite database on your machine. We have no servers, no telemetry, no cloud. The only network calls are when you explicitly ask XKG to talk to a provider (Grok, Claude, etc.) to fetch or send a message.
What does the Pro license actually unlock?
The Free version is fully usable — local storage, search, one provider. Pro unlocks the other three providers (Claude, ChatGPT, Gemini), multi-device sync between desktop and mobile, knowledge graph tagging, and lifetime updates. After buying once, you never pay again.
Can I export my data?
Anytime. The DB is a standard SQLite file you can open with any tool. You can also export to JSONL, Markdown, or HTML via the in-app export. No lock-in, ever.
What if I cancel my subscription?
Pro is a one-time purchase, not a subscription. Once you buy it, it's yours forever, including all future updates. There is no recurring charge.
How does the mobile sync work without a cloud?
XKG uses your local network (Wi-Fi) or ZeroTier/Tailscale. Your phone and desktop discover each other on the LAN and exchange data peer-to-peer. No data ever touches our infrastructure. We provide a tiny sync helper script (xkg-sync) that runs on either side.
Do I need an account?
No account, no email, no phone number. Just download and run. The only thing you'll add is your provider API keys (e.g. your x.ai key for Grok) — those are stored in your OS keyring, not on our servers.
Is the source code available?
Yes. The Tauri 2 desktop and mobile apps are on GitHub (griptoad26/xkg-desktop, griptoad26/xkg-mobile). The Pro features are compiled but verifiable against the public source.
