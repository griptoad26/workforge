👥 Task Groups Architecture

🔄 Hybrid Model: Distributed + Centralized

Each machine can work in one of three modes:

ModeDescriptionUse Case
defaultKeep tasks localPrivate work, personal tasks
localShare with groupTeam collaboration
hubSync all to HubCentral backup + viewing

📡 API Endpoints

EndpointMethodDescription
/api/task-groupsGETList all groups
/api/task-groups/:idGETGet group details
/api/task-groups/:idPUTCreate/update group
/api/task-groups/:id/membersPOSTAdd machine to group
/api/machines/:name/task-groupGETGet machine's group

🔧 How to Configure

# Add machine to a group curl -X POST http://100.112.11.35:8090/api/task-groups/web-team/members \ -H "Content-Type: application/json" \ -d '{"machine": "drone"}' # Create a new group curl -X PUT http://100.112.11.35:8090/api/task-groups/my-team \ -H "Content-Type: application/json" \ -d '{"name": "My Team", "description": "Team tasks", "members": ["x2-nuc", "drone"], "sync_mode": "local"}' # Get machine's group curl http://100.112.11.35:8090/api/machines/x2-nuc/task-group
Note: API is being finalized. The configuration file is at ~/.cluster-hub/task_groups.json
