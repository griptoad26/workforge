🪟 Phase 1: Windows Native

Terminal Access

PowerShell (Administrator) - NOT WSL

Right-click Start → Windows PowerShell (Admin)
NOT Command Prompt, NOT WSL

1 Check prerequisites:
  • PowerShell 5.1+ (pre-installed on Win10/11)
  • Python installed (python --version)
  • Tailscale installed on Windows (not WSL!)
2 Install Python if needed: 
# Download from python.org or use winget
winget install Python.Python.3.12
3 ⚠️ Tailscale MUST be on Windows side: 
# Install Tailscale in PowerShell (Admin), NOT in WSL!
winget install tailscale

# Start Tailscale
Start-Service Tailscale

# Verify
tailscale status

❌ Don't run "tailscale" in WSL - it conflicts with Windows Tailscale!

4 Create OpenClaw directory: 
New-Item -ItemType Directory -Path C:\OpenClaw -Force
cd C:\OpenClaw
5 Download OCMI modules: 
$env:CLUSTER_HUB_URL = "http://100.112.11.35:8090"

Invoke-WebRequest -Uri "$env:CLUSTER_HUB_URL/api/files/ocmi_settings.py" -OutFile ocmi_settings.py
Invoke-WebRequest -Uri "$env:CLUSTER_HUB_URL/api/files/sync_cluster_hub.py" -OutFile sync_cluster_hub.py
Invoke-WebRequest -Uri "$env:CLUSTER_HUB_URL/api/files/cross_session.py" -OutFile cross_session.py
6 Set environment variables (persistent): 
# System-wide (requires Admin)
[System.Environment]::SetEnvironmentVariable("CLUSTER_HUB_URL", "http://100.112.11.35:8090", "Machine")
[System.Environment]::SetEnvironmentVariable("OCMI_SYNC_ENABLED", "true", "Machine")

🐧 Phase 2: WSL (Windows Subsystem for Linux)

Terminal Access

WSL Ubuntu Terminal

Open Ubuntu from Start Menu (not PowerShell!)

⚠️ Important: Tailscale on Windows, Not WSL

DO NOT install Tailscale in WSL!

Install Tailscale in Windows PowerShell (Admin) instead. WSL will use Windows' Tailscale via \\wsl$\ or the Windows IP.

1 Install WSL (in PowerShell Admin): 
wsl --install

Restart computer when prompted

2 Open Ubuntu (not PowerShell!) and update: 
sudo apt update && sudo apt upgrade -y
3 Install Python and dependencies: 
sudo apt install -y python3 python3-pip python3-venv
pip3 install requests flask
4 Create directory and download modules: 
mkdir -p ~/.openclaw/memory
cd ~/.openclaw/memory

export CLUSTER_HUB_URL="http://100.112.11.35:8090"
curl -o ocmi_settings.py "$CLUSTER_HUB_URL/api/files/ocmi_settings.py"
curl -o sync_cluster_hub.py "$CLUSTER_HUB_URL/api/files/sync_cluster_hub.py"
curl -o cross_session.py "$CLUSTER_HUB_URL/api/files/cross_session.py"
5 Access Windows Tailscale from WSL: 
# WSL can access Windows network - try:
curl http://100.112.11.35:8090/health

# Or use Windows localhost proxy:
# In Windows PowerShell (Admin):
netsh interface portproxy add v4tov4 listenport=8090 connectport=8090 connectaddress=127.0.0.1

🐳 Phase 4: Docker Deployment

Dockerfile

FROM python:3.12-slim

WORKDIR /app

# Install dependencies
RUN apt-get update && apt-get install -y curl && rm -rf /var/lib/apt/lists/*

# Copy OCMI modules
COPY ocmi_settings.py sync_cluster_hub.py cross_session.py ./

# Environment
ENV CLUSTER_HUB_URL=http://100.112.11.35:8090
ENV OCMI_SYNC_ENABLED=true

EXPOSE 8080 8081 8090

CMD ["python3", "-c", "
    from sync_cluster_hub import get_sync
    import time
    sync = get_sync()
    print('Agent started')
    while True: time.sleep(60)
"]

Deploy

# Build
docker build -t openclaw-agent .

# Run
docker run -d \
  --name my-agent \
  -e CLUSTER_HUB_URL=http://100.112.11.35:8090 \
  openclaw-agent
☁️ Phase 5: VPS/Cloud Deployment

Supported Platforms

DigitalOcean
AWS EC2
Azure
Linode

VPS Install Script

#!/bin/bash
# Run as root on VPS

HUB_URL="http://100.112.11.35:8090"
AGENT_NAME="${AGENT_NAME:-vps-agent}"

# Install
apt update && apt install -y python3 python3-pip curl

# Download modules
mkdir -p /opt/openclaw
cd /opt/openclaw
curl -o ocmi_settings.py "$HUB_URL/api/files/ocmi_settings.py"
curl -o sync_cluster_hub.py "$HUB_URL/api/files/sync_cluster_hub.py"

pip3 install requests flask

# Create service
cat > /etc/systemd/system/openclaw.service << EOF
[Unit]
Description=OpenClaw Agent
After=network.target

[Service]
Type=simple
ExecStart=/usr/bin/python3 -c "from sync_cluster_hub import get_sync; get_sync().register(); import time; time.sleep(999999)"
Restart=always
Environment=CLUSTER_HUB_URL=$HUB_URL
Environment=AGENT_NAME=$AGENT_NAME

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl enable openclaw
systemctl start openclaw
