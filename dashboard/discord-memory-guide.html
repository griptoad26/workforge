💬 Discord Memory Architecture

How Memory Works Across Discord Channels

📁 Memory Files

FileScopeDescription
MEMORY.md Main Session Only Curated long-term memories - loaded in DMs only
memory/YYYY-MM-DD.md Per-Channel Raw conversation logs - separate entries per channel
AGENTS.md, SOUL.md All Sessions Workspace files - shared across all sessions

🔒 Why Separate?

Each Discord channel gets its own session to prevent conversation drift.

If channels shared context:

  • Topic A in #channel-1 would bleed into #channel-2
  • Tasks from different projects would mix
  • Context windows would overflow

🔗 How to Share Across Channels

To share important info between channels:

  1. Manually add to MEMORY.md
  2. Or use Cluster Hub to sync tasks
  3. Or DM the main session with updates
